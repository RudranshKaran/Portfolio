## Image Specifications

- **Format**: JPG or WebP (for best Next.js optimization)
- **Resolution**: 2x actual display size for retina displays
- **Optimization**: Compress images with tools like:
  - [TinyPNG](https://tinypng.com)
  - [Squoosh](https://squoosh.app)
  - [ImageOptim](https://imageoptim.com)
